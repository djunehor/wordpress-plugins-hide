ZacWP Hide Plugins allows a website administrator hide plugins that shows in the plugins page.

== Description ==
This plugin is useful for developers that have used certain custom plugins for client and want to prevent client from accidentally disabling or even deleting a plugin that's cogent to the website.

* Under Settings, you'll see plugin settings
* Simply select the plugins to hide and save

== Installation ==

1. Upload the entire `zacwp-plugin-hide` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
